Run:

```sh
likec4 export png --views "overview*" --theme dark --output ./images --flat
```

If you run it from the current LikeC4 project directory, the project path is optional and you can omit it as shown above.
If you want to make the project path explicit, use:

```sh
likec4 export png --project . --views "overview*" --theme dark --output ./images --flat
```
