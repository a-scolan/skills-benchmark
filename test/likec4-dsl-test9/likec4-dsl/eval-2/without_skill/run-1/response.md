Run:

`likec4 export png --views "overview*" --theme dark --output ./images --flat`

If you are already in the current project directory, the project path is optional and can be omitted as shown above.
If you are **not** in the project directory, append the project path as the final argument, for example:

`likec4 export png --views "overview*" --theme dark --output ./images --flat .`
